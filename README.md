# TianJi 天玑系统

**东方象数增强型多智能体推演引擎**

融合八卦、五行、四柱、奇门、易经与传统推演逻辑，预测未来路径，复盘过去节点，给出行动建议。

---

## 核心能力

**不只是输出术语，而是给出可以执行的判断。**

系统包含完整的象数引擎（八字、奇门、易经）和一个专门的通俗解读层——把专业术语翻译成用户能看懂的一句话结论和可执行的动作建议。

## 快速开始

### 命令行

```bash
cd backend
pip install -r requirements.txt
PYTHONPATH=. python scripts/tianji_cli.py \
  "我想追这个女生，后续怎么推进？" \
  --domain relationship \
  --goal "判断关系推进路径" \
  --event-time "2026-05-26 22:00" \
  --birth-datetime "1998-06-15 14:30" \
  --gender male
```

### API

```bash
curl -X POST http://localhost:5000/api/tianji/run \
  -H "Content-Type: application/json" \
  -d '{
    "question": "这个项目适不适合公开发布？",
    "domain": "strategy",
    "goal": "判断发布时机",
    "event_time": "2026-05-26 22:00",
    "birth_datetime": "1998-06-15 14:30"
  }'
```

## 报告结构

推演完成后，报告包含 9 个章节：

```
1. 现实解析：事实清单 + 时间线 + 已知/未知变量
2. 因果时序：过去节点的反推
3. 概率分支：未来三条路径（A/B/C）
4. 行动建议：最佳动作 + 禁忌 + 观察信号
5. 四柱分析：六十甲子 + 月令节气 + 十神 + 藏干 + 用神忌神
6. 奇门分析：值符值使 + 九宫飞布 + 八门/九星/八神
7. 易经分析：主卦 + 错卦 + 互卦 + 动爻 + 阶段提示
8. 置信度与盲区：未知变量统计 + 判断原则
9. 通俗解读：用户能看懂的一句话说结论 + 分层行动建议（新）
```

**第 9 章通俗解读示例：**

```markdown
## 9. 通俗解读

### 一句话结论
有机会但有隐情，需要试探验证。顺势而为

### 四柱通俗解读
- 日主癸，水属性。灵活变通但容易犹豫不决。月令午月。
- 身强：你目前状态偏弱，底气不足，不适合强推
- 用神：天干庚,戊 适合你，用温和方式给对方留空间
- 忌神：天干癸,壬 对你不利

### 奇门通俗解读
- 当前局势阳遁4局，主坎一宫，大开，机会现
- 主星声量和表现，容易虚火过旺
- （注：巳,午宫空亡，存在虚假信号，不要过度解读）

### 易经通俗解读
- 当前主卦泽山咸。趋势围绕感应，关键在吸引与互动
- 建议：围绕感应小步验证
- 风险：忽视吸引导致失衡

### 综合信号
- 核心主题：机会现、信号强、宜守
- 整体基调：有机会但有隐情，需要试探验证

### 行动建议
你目前状态偏弱，不适合强推。奇门开门有势，可以主动一点但要看反馈。
三路径：
- A 顺势发展（45%）：对方回应变积极 → 继续聊，保持自然
- B 中途受阻（35%）：对方依然冷淡 → 减少主动，观望等待
- C 突发反转（20%）：对方态度突变 → 准备撤退或调整策略
观察：看对方是否持续回应 / 看关键节点是否有变化 / 看风险是否变明显
```

## 反馈回填

每次推演后，将实际结果回填到系统，系统会积累案例并统计准确率。

```bash
curl -X POST http://localhost:5000/api/tianji-review/reviews/<report_id> \
  -H "Content-Type: application/json" \
  -d '{
    "feedback": "实际发生了XX，结果是XX",
    "accuracy": 3,
    "notes": "预测偏差在哪里"
  }'
```
