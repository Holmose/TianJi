# TianJi 天机推演系统

**一个用东方哲学做现实推演的工具**

把你想推演的事情、背景、人物信息丢进来，系统会结合八卦、五行、易经、四柱、奇门，给你一套多角度的分析报告，帮你看清楚现状、判断未来可能的走向、找到下一步该做什么。

---

## 能做什么

输入一段问题或事件背景，系统会输出：

- 当前局势属于什么象（乾主动、坎有风险、离要曝光……）
- 五种元素的强弱分布（你现在是木太多还是火不够）
- 易经卦象变化趋势（本卦是什么、变卦是什么）
- 人物的四柱结构（性格倾向、风险模式）
- 奇门局势（现在时机好不好、该进还是该退）
- 多角度推演结论（3条未来路径 + 1条黑天鹅路径）
- 当前最该做的1件事，以及最不该做的几件事

## 快速开始

### 1. 安装依赖

```bash
npm run setup
```

### 2. 运行一次推演

```bash
cd backend
PYTHONPATH=. python scripts/tianji_cli.py \
  "这个项目还没完全成熟，未来适不适合公开发布？" \
  --domain strategy \
  --event-time "2026-05-26 22:10" \
  --birth-datetime "1998-06-15 14:30" \
  --out reports/tianji
```

输出的报告在 `reports/tianji/` 目录。

### 3. 运行测试

```bash
npm run test
```

## 输入参数说明

| 参数 | 必填 | 说明 |
|---|---|---|
| `question` | 是 | 你想推演的问题或事件背景 |
| `domain` | 否 | 领域：`relationship` / `business` / `content` / `personal` / `strategy` |
| `event_time` | 否 | 事件时间，用于奇门局势分析 |
| `location` | 否 | 地点 |
| `birth_datetime` | 否 | 出生时间，用于四柱人物分析，格式 `YYYY-MM-DD HH:MM` |
| `gender` | 否 | 性别 |
| `rounds` | 否 | 推演轮数，默认3 |

## 报告长什么样

推演结束后，你会得到一份报告，核心内容如下：

```
## 1. 总断
主象为 乾/金，趋势参考 天火同人→地泽临
建议：可主动定方向，但避免压迫

## 2. 现实底盘
- 项目还没完全成熟
- 有一定用户基础
- 资金还在投入期

## 3. 象数建模
- 八卦：乾，主动、权力、目标
- 五行：金偏旺，过硬过冷是风险
- 四柱：日主癸水，擅长观察变化，但金偏弱是短板
- 奇门：阳遁9局，景门主事，时机偏开

## 4. 未来路径
### A 顺势发展｜45%
触发条件：现实反馈持续增强
### B 中途受阻｜35%
触发条件：信息不足扩大、外部变量介入
### C 突发反转｜20%
触发条件：隐藏变量暴露

## 5. 当前最优动作
- 该做：主动定方向，小步推进
- 不该做：压迫式发布、过度承诺、跳过现实反馈
- 观察信号：用户反馈、数据变化、外部竞争
```

---

## 适合谁用

- 想系统化思考复杂决策的人
- 对东方哲学和决策推演感兴趣的人
- 需要多角度分析但不想要玄学迷信的人

**系统不会给你一个绝对答案，而是给你3条路径 + 触发条件 + 风险提示，让你自己判断。**

---

## 注意事项

- 推演是辅助工具，不是预测神器
- 投资、医疗、法律等高风险决策，请咨询专业人士
- 系统会保留盲区说明，不会假装什么都知道
